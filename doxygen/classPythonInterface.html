Detailed Description

Interface for receiving Hamiltonian data from Python.

This class provides functionality to read system and control Hamiltonians from files generated by Python scripts. It supports both Lindblad and Schroedinger solvers and handles matrix data communication between Python and the C++ Quandary core.
