Detailed Description

Base class for quantum gate operations.

The Gate class provides the fundamental interface for quantum gate operations in quantum optimal control. It handles the transformation V \rho V^\dagger where V is the unitary gate matrix and \rho is the quantum state density matrix.

void Gate::assembleGate ( )

Assembles the vectorized gate matrices.

Computes VxV_re = Re(\bar V \kron V) and VxV_im = Im(\bar V \kron V) where V is the gate matrix and \bar V is its complex conjugate.
